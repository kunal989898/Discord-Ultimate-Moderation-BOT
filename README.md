
# Null • Core — Professional Edition

Enterprise-grade Discord moderation bot with guided UI.

## Setup
1. Node.js 18+
2. npm install
3. Set BOT_TOKEN
4. npm start

## Features
- Olympus-style help dashboard
- Fully explained mass moderation
- Safety-first execution
- Persistent settings
- UI + text fallback

## Commands
!help, !ping
!masskick, !massban

!massbanrole @Role
